# Productividad: Mi setup de desarrollo

Como desarrollador, las herramientas que usamos tienen un gran impacto en nuestra productividad. Aquí comparto mi setup actual.

## Hardware

### Computadora
- **Procesador**: AMD Ryzen 7 / Intel i7
- **RAM**: 16GB DDR4
- **Monitor**: 27" 1440p
- **Teclado**: Mecánico (switches Cherry MX)
- **Mouse**: Logitech MX Master 3

## Software

### Editor de código
**Visual Studio Code** es mi editor principal. Algunas extensiones favoritas:

- Prettier - Code formatter
- ESLint
- GitLens
- Auto Rename Tag
- Material Icon Theme
- GitHub Copilot

### Terminal
Uso **Windows Terminal** con PowerShell 7 y:
- Oh My Posh para personalización
- Comandos personalizados (aliases)
- Git integrado

### Navegadores
- **Chrome** para desarrollo (DevTools son excelentes)
- **Firefox** para testing
- **Edge** ocasionalmente

## Flujo de trabajo

### Morning routine
1. Revisar emails y mensajes
2. Planificar tareas del día
3. Sesión de código enfocada (2-3 horas)
4. Break

### Herramientas de productividad
- **Notion**: Notas y documentación
- **Todoist**: Gestión de tareas
- **Spotify**: Música de concentración
- **Pomodoro Timer**: Sesiones de 25 minutos

## Consejos para mejorar productividad

1. **Automatiza tareas repetitivas**: Scripts, snippets, shortcuts
2. **Toma breaks regulares**: Previene burnout
3. **Aprende shortcuts**: Ahorra mucho tiempo
4. **Mantén el workspace limpio**: Menos distracciones

## Conclusión

Tu setup ideal depende de tus necesidades y preferencias. Lo importante es encontrar lo que funciona para ti y optimizarlo continuamente.
